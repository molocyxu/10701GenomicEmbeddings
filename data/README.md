gene_sequences_12kb_4912genes.csv is a CSV file containing 4912 gene region sequences.
It has two main columns: 1. gene_id: The Ensembl gene ID for each gene; 2. sequence: The corresponding 12kb (TSS-centered) DNA sequence for each gene_id.
